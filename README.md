# React Form Submission App

A simple React application that demonstrates form handling using controlled components and React Hooks. This project includes various form elements such as text inputs, radio buttons, checkboxes, file uploads, dropdowns, and textareas, with all form data managed through React state.


## Features

* Controlled form inputs using `useState`
* First Name and Last Name fields
* Email and Contact inputs
* Gender selection with radio buttons
* Multiple subject selection using checkboxes
* Resume file upload
* URL input field
* Select dropdown with grouped options
* About Yourself textarea
* Form submission handling
* Form reset functionality
* Clean component-based structure


## Technologies Used

* React
* JavaScript (ES6+)
* CSS
* Vite

## Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project folder

```bash
cd form-submission-app
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```
